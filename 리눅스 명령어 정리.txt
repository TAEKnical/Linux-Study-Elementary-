*****리눅스 명령어 정리****
Lecture1
mkdir : 디렉토리 생성
pwd : 현재 디렉토리 경로 출력
ls : 현재 디렉토리 파일,폴더 목록
ls -l : 현재 디렉토리 파일 상세 출력
	ls -l 입력시 맨 앞이 d이면 디렉토리

Lecture2
cd 디렉토리경로 : 해당 디렉토리로 이동
cd .. : 이전(부모)디렉토리로 이동
rm 파일명 : 파일삭제
rm -r 디렉토리 : 디렉토리삭제
-help 옵션 : 명령어 사용법, 메뉴얼
--help 옵션 :  옵션 설명(ex: rm -help  하면 rm과 관련한 각종 옵션명령어에대한 도움말)

Lecture3

man 명령어 : 명령어 사용설명서 출력(help는 간소화 메뉴얼, man은 상세 메뉴얼)
	man 명령어 사용중에 / 내용 을 입력하면 일치하는 텍스트를 찾는다. n키를 누르면 다음
	텍스트 위치로 넘어감.
ls --all : 감춰진 파일 포함 모든 파일의 정보 상세 출력

ls -al , ls -la : a(감춰진 파일 포함) 와 l(상세정보) 옵션 모두 적용
ls -alS : 위와 같으며 정렬해서 출력

Lecture4

cp 원본파일위치_및_파일이름 목적지위치_및_파일이름 : 파일 복사
mv 원본파일위치-및_파일이름 목적지위치_및_파일이름 : 파일 이동
ls -l 디렉토리_또는_파일이름 : 해당 디렉토리 또는 파일 정보 상세출력
mv name1.txt name2.txt : 파일 이름 변경(mv의 응용)

Lecture5
sudo(super user do)
명령어 앞에 sudo를 붙여주면 root의 권한으로 명령어를 실행하게 됨
